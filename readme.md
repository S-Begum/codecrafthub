A simple REST API built with Node.js and Express to help developers track their learning goals. Perfect for beginners learning REST API concepts!

## 📋 Project Overview

CodeCraftHub is a lightweight API that allows developers to create and manage their personal learning courses. Instead of using a complex database, it stores all data in a simple JSON file, making it perfect for learning and small projects.

## ✨ Features

- **Complete CRUD Operations**: Create, Read, Update, and Delete learning courses
- **JSON File Storage**: No database required - simple and portable
- **Data Validation**: Ensures all course data meets requirements
- **Automatic ID Generation**: Each course gets a unique ID
- **Timestamps**: Auto-generated creation and update timestamps
- **Status Tracking**: Track courses as "Not Started", "In Progress", or "Completed"
- **Error Handling**: Comprehensive error messages for common issues
- **Beginner-Friendly**: Well-commented code perfect for learning
